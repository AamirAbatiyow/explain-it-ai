# explain-it-ai
Here is 4stacks submission for the CCSC Hackathon.

After opening the GitHub repository link, click the green “Code” button near the top right of the file list. In the dropdown, click “Download ZIP”, and the repository will download to your computer as a .zip file. Once downloaded, locate the file in your Downloads folder and double-click it to extract the contents. You can then open the extracted folder and access the project files locally.
